# Diem Plugin (portable)

1. Give this entire folder to your agent.
2. Tell it to run:

```bash
bash /path/to/diem-plugin-share/scripts/install.sh
```

3. Restart OpenClaw gateway.
4. Use `/diem`.

Notes:
- No secrets are included in this folder.
- Plugin reads Venice auth from local OpenClaw auth profiles on the target machine.
